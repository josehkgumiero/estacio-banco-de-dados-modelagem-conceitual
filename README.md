# Banco de Dados Modelagem Conceitual
O projeto está disponível em: [Projeto de Banco de Dados: Modelagem Conceitual](Documentation/Projeto_de_Banco_de_Dados_Modegem_Conceitual.docx)
# Resumo
Este projeto apresenta os fundamentos do desenvolvimento de um banco de dados, com foco na modelagem conceitual e nas etapas que compõem o processo de projeto de banco de dados. O material aborda o levantamento de requisitos, a construção do Diagrama Entidade-Relacionamento (DER), a identificação de entidades, relacionamentos, atributos e cardinalidades, além da evolução para os projetos lógico e físico. Também são explorados conceitos como atributos simples e compostos, especialização/generalização, entidade associativa e agregação, demonstrando como esses elementos contribuem para representar corretamente as regras de negócio e servir de base para a implementação de bancos de dados relacionais.
# Conclusão
O estudo do projeto de banco de dados evidencia que a modelagem conceitual é uma etapa essencial para transformar os requisitos do negócio em uma representação estruturada e compreensível das informações. A elaboração do Diagrama Entidade-Relacionamento permite identificar entidades, atributos, relacionamentos e cardinalidades, reduzindo ambiguidades e garantindo que as regras de negócio sejam representadas de forma consistente antes da implementação do banco de dados. A continuidade do processo por meio dos projetos lógico e físico demonstra como um modelo conceitual bem definido é convertido em tabelas, relacionamentos e comandos SQL específicos para um SGBD. Além disso, a aplicação de conceitos como especialização/generalização, entidades associativas e diferentes tipos de atributos torna o modelo mais flexível, organizado e aderente às necessidades do sistema. Dessa forma, o projeto reforça a importância da modelagem de dados como base para o desenvolvimento de bancos de dados confiáveis, escaláveis e de fácil manutenção, proporcionando uma documentação técnica que facilita a comunicação entre analistas, desenvolvedores e demais envolvidos no projeto.
